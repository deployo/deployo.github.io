Overview

Deployo is a web based shell execution tool that allows you to execute over SSH with a button click. It is perfectly suited for personal projects, freelancers or small companies looking to speed up their remote tasks.

Features

Execute shell scripts

Run your scrips easily with live output streamed directly from your server.

See history

History of all your script executions is shown right on your dashboard. You can access old executions and execute again the same script with same parameters with ease.

Configure easily with YAML

All your configuration is stored online in a simple YAML file. Easily edit with syntax highlighting.

Access from anywhere

Deployo is a web based solution accessible from any device with a browser. So desktop, tablet or mobile, it is at your fingertips.

Store your SSH keys

Your SSH keys can be generated and stored online and used everytime you acces your server. Have a private key already? Just upload it!
